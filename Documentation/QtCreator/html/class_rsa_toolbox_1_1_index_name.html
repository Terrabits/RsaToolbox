Detailed Description

The IndexName class handles identifiers that include both an index and a name.

IndexName is primarily a data type for resources that have both a name and a label as identifiers. For example, Rohde & Schwarz Vector Network Analyzers identify channels by both an index number and a name.

Note: &IndexName is case-sensitive.
