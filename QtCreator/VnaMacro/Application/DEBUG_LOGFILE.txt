
Log data is placed here when compiled in debug mode.

Release logfile can be found in the appropriate installation directory for your system.

For Windows 7, this is typically:
C:\ProgramData\Rohde-Schwarz\%APP_FOLDER%\%LOG_FILENAME%

For Windows XP, this is typically:
C:\Documents and Settings\All Users\Application Data\Rohde-Schwarz\%APP_FOLDER%\
